# 🚀 SAP O2C Capstone Project (Order-to-Cash)

## 📌 Overview
This project demonstrates the complete **Order-to-Cash (O2C)** cycle using SAP SD concepts and a Python-based API simulation.

Based on SAP ERP implementation for:
**NovaTex Industries Pvt. Ltd.**

📄 Full Report: Included in `/docs`

---

## 🔄 O2C Process Flow

1. Inquiry (VA11)
2. Quotation (VA21)
3. Sales Order (VA01)
4. Delivery (VL01N)
5. Post Goods Issue (VL02N)
6. Billing (VF01)
7. Payment (F-28)

- Python
